# 🛒 Superstore Sales Intelligence Dashboard

A professional 3-tab sales analytics dashboard built with Python and Streamlit.
Analyzes 9,994 orders across 4 years, 4 regions, and 3 product categories.

## 🚀 Live Demo
[➡ Open App on Streamlit Cloud](#) ← replace with your deployed URL

## 📊 What's Inside

### Tab 1 — Overview
- Monthly Sales & Profit trend
- Sales by Category (grouped bar)
- Sales & Profit by Region
- Quarterly performance
- Customer segment breakdown
- 5 auto-generated business insights

### Tab 2 — Products
- Profit vs Sales scatter (spot loss-makers instantly)
- Discount impact on profit (does discounting hurt?)
- Top 10 products by sales
- Loss-making orders table

### Tab 3 — Customers & Shipping
- Ship mode breakdown with avg delivery days
- Days-to-ship distribution
- Top 15 customers by sales
- Top 10 states by revenue

## ⚡ Run Locally

```bash
git clone https://github.com/YOUR_USERNAME/superstore-dashboard.git
cd superstore-dashboard
pip install -r requirements.txt
streamlit run app.py
```

## ☁️ Deploy Free on Streamlit Cloud

1. Push to GitHub
2. Go to share.streamlit.io
3. Connect repo → main file: app.py
4. Deploy ✅
